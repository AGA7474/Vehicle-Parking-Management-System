 
# Vehicle Parking System

A simple console-based application built in **C#** that manages vehicle parking operations such as vehicle registration, slot allocation, ticket generation, and fee calculation.

## Features
- Vehicle registration and slot management  
- Ticket generation and parking fee calculation  
- Console-based interface for easy use  
- Built using **Object-Oriented Programming (OOP)** concepts:
  - Classes & Objects  
  - Inheritance  
  - Encapsulation  
  - Polymorphism

## Technologies
- C#  
- .NET Framework / .NET SDK  
- Visual Studio

## Usage
1. Open the project in Visual Studio  
2. Build and run the solution (`.sln` file)  
3. Follow the console instructions to test parking operations
